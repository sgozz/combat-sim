# Mobile UX Fixes - Decisions

## Architecture Decisions

### Target Selection (Task 2)
- **Decision**: Make initiative tracker tappable (not new component)
- **Rationale**: Simplest solution, no new UI needed

### Camera Orientation (Task 3)
- **Decision**: Auto-center on turn change (not minimap)
- **Rationale**: User preference, less UI clutter

### Test Strategy
- **Decision**: Tests-after (implement first, test later)
- **Rationale**: Faster iteration for UI fixes
